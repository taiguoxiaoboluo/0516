# Style Sniffer 截图画廊

这些截图来自 `测试/Style Sniffer 🐕_🦺 · 风格嗅探器.html`，用于快速说明 Style Sniffer 的界面、输入方式和结果形态。

## 总览

![Style Sniffer 界面总览](./readme-assets/style-sniffer-overview.png)

## 输入区

支持网页 URL、图片 / 截图、已有 JSON 三种入口。

![输入方式](./readme-assets/style-sniffer-input.png)

## Prompt 结果

嗅探完成后会生成中英文双栏 Prompt，可直接复制给 AI Agent 或页面生成流程。

![生成的 Prompt](./readme-assets/style-sniffer-generated-prompt.png)

## 色彩 Token

将页面中的视觉颜色归纳成背景、前景、主色、次色、强调色、边框色等 Token。

![色彩 Token](./readme-assets/style-sniffer-colors.png)

## 风格画像

把页面气质归纳为风格标签、类型、复杂度、留白、对比度和装饰程度。

![风格画像](./readme-assets/style-sniffer-style-profile.png)

## 溯源证据

逐屏展示页面截图和对应 CSS / DOM 风格特征，方便检查判断依据。

![溯源证据](./readme-assets/style-sniffer-evidence.png)

# 给别人本地运行的超简单说明

这份说明给不懂代码的人看。

你可以把这个工具理解成一个小软件。  
如果只想上传图片分析，可以直接用在线网页。  
如果想输入一个网页地址，让工具读取那个网页的 HTML / CSS 并分析风格，就需要在自己的电脑上本地运行一次。

## 你需要先准备什么

电脑里需要安装 Node.js。

点击这里安装：[Node.js](https://nodejs.org)

进入网站后，选择 LTS 版本下载并安装即可。

## 方式一：从 GitHub 下载

打开这个地址：[https://github.com/taiguoxiaoboluo/0516](https://github.com/taiguoxiaoboluo/0516)

点击绿色的 **Code** 按钮，然后点 **Download ZIP**。

下载完成后，把压缩包解压。

也可以直接点这个下载项目包：[Download ZIP](https://github.com/taiguoxiaoboluo/0516/archive/refs/heads/main.zip)

## 方式二：别人直接发你压缩包

如果别人已经把项目压缩包发给你，直接解压即可。

## 启动方法

解压后，打开项目文件夹。

然后打开终端。

如果你用的是 Mac：

1. 打开项目文件夹
2. 在文件夹空白处右键
3. 选择“新建位于文件夹位置的终端窗口”

如果你用的是 Windows：

1. 打开项目文件夹
2. 在地址栏输入 `cmd`
3. 按回车

然后依次运行下面两行命令。

第一行：

```bash
npm install
```

这一步是在安装工具需要的依赖。第一次运行会慢一点。

第二行：

```bash
npm run web
```

这一步是在启动工具。

## 打开网页

启动成功后，终端里通常会出现类似这样的地址：

```text
http://127.0.0.1:3000
```

打开浏览器，把这个地址复制进去。

然后你就可以使用工具了。

## 哪些功能可以用

本地运行后，可以使用：

- 上传图片 / 截图分析
- 输入网页 URL 分析
- 读取网页 HTML / CSS 风格
- 生成 JSON
- 生成 Prompt
- 生成 CSS Token

## 常见问题

### 为什么在线网页不能读取 HTML？

因为在线网页只是一个普通网页，它不能替你打开别的网站并读取 HTML。

读取网页 URL / HTML 需要电脑里启动一个小服务。  
运行 `npm run web` 就是在启动这个小服务。

### 关闭后还能用吗？

如果终端窗口关掉了，工具也会停止。

下次再用时，重新打开项目文件夹，然后运行：

```bash
npm run web
```

再打开：

```text
http://127.0.0.1:3000
```

### 需要每次都运行 npm install 吗？

不需要。

`npm install` 通常只需要第一次运行。  
以后只需要运行：

```bash
npm run web
```

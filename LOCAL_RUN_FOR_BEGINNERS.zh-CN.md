# 给别人本地运行的超简单说明

这份说明给不懂代码的人看。

只上传图片 / 截图，可以直接用在线网页。

要输入网页 URL 并读取 HTML / CSS，就需要在自己的电脑上启动一次本地小服务。

## 第一次只做三步

### 1. 下载两个东西

先点这里安装：[Node.js 安装包](https://nodejs.org)

进入后选择 **LTS** 版本，下载并安装。这个只需要安装一次。

再点这里下载：[项目包](https://github.com/taiguoxiaoboluo/0516/archive/refs/heads/main.zip)

下载后把压缩包解压。

### 2. 在项目文件夹里复制两行命令

打开解压后的项目文件夹，然后打开终端。

Mac：在文件夹空白处右键，选择“新建位于文件夹位置的终端窗口”。

Windows：在文件夹地址栏输入 `cmd`，然后按回车。

在终端里依次复制运行：

```bash
npm install
npm run web
```

第一行是第一次准备工具，可能会慢一点。

第二行是启动本地网页。

### 3. 回到浏览器打开本地网页

打开：

```text
http://127.0.0.1:3000
```

然后就可以输入网页 URL 分析 HTML / CSS 了。

## 以后再用

以后不用重新安装 Node.js，也通常不用再运行 `npm install`。

只要：

1. 打开项目文件夹
2. 打开终端
3. 运行：

```bash
npm run web
```

再打开：

```text
http://127.0.0.1:3000
```

## 一句话版本

```text
先安装 Node.js，再下载项目包；在项目文件夹里复制运行 npm install 和 npm run web；最后打开 http://127.0.0.1:3000。
```

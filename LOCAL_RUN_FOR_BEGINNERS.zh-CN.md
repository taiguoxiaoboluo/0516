# 给别人本地运行的超简单说明

这份说明给不懂代码的人看。

只上传图片 / 截图，可以直接用在线网页。

要输入网页 URL 并读取 HTML / CSS，请使用作者发你的免安装本地工具包。

## 第一次只做三步

### 1. 解压工具包

拿到工具包后，先解压。

这个工具包已经自带运行环境，不需要另外安装任何东西。

### 2. 双击启动

Mac 版里会有这个文件：

```text
启动 Style Sniffer.command
```

双击它。

它会打开一个终端窗口，然后自动打开浏览器。

如果 Mac 提示不能打开，就按住 control 键点击这个文件，再选择“打开”。

### 3. 粘贴网页 URL

浏览器打开后，粘贴你想分析的网页地址，然后点击开始。

如果浏览器没有自动打开，就手动打开：

```text
http://127.0.0.1:3000
```

## 以后再用

以后还是双击：

```text
启动 Style Sniffer.command
```

用完以后，关闭终端窗口即可。

本地小服务

分析网页 URL / HTML：下载后复制两行命令

1

先下载两个东西：点 Node.js 安装包 安装一次；再点 项目包 下载并解压。

2

让终端进入项目文件夹，然后复制运行：

Mac：打开「终端」App，输入 cd 和一个空格，把项目文件夹拖进终端，按回车。Windows：在项目文件夹地址栏输入 cmd 回车。

npm install
npm run web
3

回到浏览器打开：http://127.0.0.1:3000，然后就能粘贴网页 URL 分析 HTML / CSS。

嗅探设计风格

网页 URL 怎么用

本地运行后，在这里粘贴网页地址，例如 https://example.com，工具会打开页面并读取 HTML / CSS 风格。

把网页截图、设计稿或参考图片拖进来，点击「开始嗅探」，即可得到颜色、风格 Prompt 和 CSS Token。

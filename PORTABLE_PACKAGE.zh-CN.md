# 免安装本地工具包

这个方案的目标是：别人不用额外安装任何运行环境。

你先在自己的电脑上打包一次，生成一个自带运行环境的本地工具包。别人拿到后只需要解压、双击启动。

## 给普通用户的用法

1. 解压工具包
2. 双击 `启动 Style Sniffer.command`
3. 浏览器自动打开后，粘贴网页 URL

如果浏览器没有自动打开，手动打开：

```text
http://127.0.0.1:3000
```

如果 Mac 提示不能打开：

按住 control 键点击 `启动 Style Sniffer.command`，再选择“打开”。

## 给作者的打包方法

在项目目录里运行：

```bash
npm run package:mac
```

生成位置：

```text
dist/Style Sniffer 本地工具-mac-arm64
```

这个文件夹就是免安装工具包。

## 为什么包会比较大

因为它把这些东西都放进去了：

- Style Sniffer 工具代码
- Node 运行环境
- 用来读取网页的浏览器内核

这样普通用户不用再单独安装任何运行环境。
